PvP Training Pack
=================

This modpack is explicitly designed for practicing and mastering advanced Minecraft PvP techniques, with a special focus on the Mace weapon and advanced movement.

WHAT DOES IT INCLUDE?
---------------------

*   **Carpet PvP Tools:** Dedicated mods like Carpet and its PvP extensions for spawning and managing training bots.
*   **Elytra Training:** Includes mods to facilitate practice for advanced Elytra-Macing and combat movements.
*   **Voice Chat Mod:** Integrated voice communication for training with friends.
*   **Training World:** A custom-built map specifically designed for combat drills (included in the main pack file).

UNIQUE FEATURES
---------------

*   The first publicly available modpack dedicated to structured Mace and high-level PvP training.
*   This version (2.0.0) features a significantly improved custom training world and optimized mod configuration.

ADVANCED BOT COMMANDS (In-Game Reference)
-----------------------------------------

For players using the bot named **TheobaldTheBot**, here is a quick guide to restoring gear and using advanced commands:

**Q: What if the bot's armor breaks?**

**A:** Use these steps to restore and re-equip the gear (Mending is included, but this forces a quick reset):

1.  **Give New Gear (replace `<slot>` with the inventory slot number where the item lands):**
    
    *   **Sword:** `/give TheobaldTheBot minecraft:netherite_sword[minecraft:enchantments={levels:{"minecraft:sharpness":5,"minecraft:unbreaking":3,"minecraft:fire_aspect":2,"minecraft:mending":1}}]`
    *   **Helmet:** `/give TheobaldTheBot minecraft:netherite_helmet[minecraft:enchantments={levels:{"minecraft:protection":4,"minecraft:unbreaking":3,"minecraft:mending":1}}]`
    *   **Chestplate:** `/give TheobaldTheBot minecraft:netherite_chestplate[minecraft:enchantments={levels:{"minecraft:protection":4,"minecraft:unbreaking":3,"minecraft:mending":1}}]`
    *   **Legs:** `/give TheobaldTheBot minecraft:netherite_leggings[minecraft:enchantments={levels:{"minecraft:protection":4,"minecraft:unbreaking":3,"minecraft:mending":1}}]`
    *   **Boots:** `/give TheobaldTheBot minecraft:netherite_boots[minecraft:enchantments={levels:{"minecraft:protection":4,"minecraft:unbreaking":3,"minecraft:mending":1}}]`
2.  **Equip the Item:**
    
    *   `/player TheobaldTheBot hotbar <slot>`
    *   `/player TheobaldTheBot use once`

**Q:** **How to add an Off-Hand Item?**

**A:** **Select the slot the item is in and use:** `/player TheobaldTheBot swapHands`
